# Circuit Breaker: Defusal Unit

**Code:** XCB  
**Remote:** https://github.com/Lexxicc/Circuit-Breaker.git  
**Version:** v0.0.6 (`Circuit Breaker v0.0.6.html`)  
**Genre:** Puzzle / bomb defusal

## Tech
- DOM-based rendering (no canvas)
- Vanilla JS + Tailwind CSS via CDN
- Single-file HTML — no build step

## localStorage
- `circuit_breaker_high` — high score
- `circuit_breaker_history` — game history log
- `circuit_breaker_stats` — player stats

## Notes
- Keys use `circuit_breaker_` prefix, not `XCB_` — note the non-standard naming
